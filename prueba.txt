Perfecto, te explico cada paso de forma simple y rápida para que puedas hacerlo vos, y si querés, te ayudo con los comandos o incluso generarte capturas de ejemplo. Vamos:


---

1. Instalación de Git

Verificá si ya está instalado: Abrí la terminal (CMD o PowerShell en Windows, o Terminal en Linux/macOS) y escribí:

git --version

Si te muestra una versión como git version 2.43.0, ya lo tenés.

Si no, instalalo desde: https://git-scm.com/downloads (elegí tu sistema operativo, descargá e instalá).



---

2. Configuración inicial de Git

Ejecutá estos comandos con tus datos:

git config --global user.name "Tu Nombre"
git config --global user.email "tunombre@ejemplo.com"

Podés verificar si quedó bien con:

git config --global --list


---

3. Crear un repositorio local

mkdir prueba-git
cd prueba-git
git init

Con eso ya tenés un repo vacío.


---

4. Seguimiento de archivos

echo "# Mi primer repo con Git" > README.md
git add README.md
git commit -m "Primer commit: agrego README"


---

5. Conectar con GitHub

1. Andá a https://github.com y creá un repositorio nuevo (sin README).


2. Copiá la URL que te da (será algo así como https://github.com/tuusuario/prueba-git.git).


3. En la terminal:



git remote add origin https://github.com/tuusuario/prueba-git.git
git branch -M main
git push -u origin main

Si es tu primer push, puede pedirte que inicies sesión en GitHub desde la terminal o con un token.


---

6. Hacer cambios y nuevos commits

Editá el archivo README.md (agregá una línea o texto nuevo), y luego:

git add README.md
git commit -m "Actualizo el README con info nueva"
git push
